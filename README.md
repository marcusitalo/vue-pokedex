I also used LocalStorage to decrease the number of requests and provide quick navigation.

## Clone repository
```
git clone https://github.com/marcusitalo/vue-pokedex.git
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```
